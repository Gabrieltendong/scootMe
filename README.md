# ScootMe

## Clone app 
```bash
# using git
$ git clone https://github.com/Gabrieltendong/foodBin.git
```

NB: after clone rename scootMe.text to scootMe.jks

## Install dependency

```bash
# using yarn
$ yarn install

# using npm
$ npm install
```

## Running app

```bash
$ expo start
```

## Build release app

```bash
# install eas cli
$ npm install -g eas-cli

# using eas to build app
$ eas build --platform android
```
